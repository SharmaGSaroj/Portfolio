## 🛠 Installation & Development

```bash
# Clone the repository
git clone https://github.com/SharmaGSaroj/Portfolio.git

# Navigate into the directory
cd Portfolio

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
